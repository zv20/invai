## ðŸš€ Getting Started

### First Time Setup
1. **Set Reorder Points** - Go to Inventory, edit products, set reorder thresholds
2. **Favorite Products** - Star your most-used items
3. **Try Dark Mode** - Click the moon icon in header
4. **Learn Shortcuts** - Press `?` to see all keyboard shortcuts
5. **Explore Reports** - Click Reports tab to see analytics

### Power User Tips
1. Use `Ctrl+K` for instant access to any action
2. Use `Ctrl+F` to jump to search without clicking
3. Navigate tabs with `Ctrl+1/2/3/4` instead of clicking
4. Press `ESC` to quickly close any modal or clear search
5. Check Activity Feed daily to track team changes

## ðŸ“Š Reports Explained

### Stock Value Report
- Shows total inventory worth
- Breaks down by category and supplier
- Helps with financial planning
- Export to CSV for accounting

### Expiration Report
- **Expired** - Remove immediately
- **Urgent (7 days)** - Use or discount ASAP
- **Soon (30 days)** - Plan promotions
- Export for waste tracking

### Low Stock Report
- Shows products below reorder point
- Indicates how many units needed
- Use to create purchase orders
- Export for supplier communication

## ðŸ› Troubleshooting

### Dark mode not working?
- Refresh the page
- Check Settings > Display > Theme is set correctly

### Keyboard shortcuts not working?
- Make sure you're not in an input field
- On Mac, shortcuts use `Cmd` instead of `Ctrl`
- Press `?` to verify shortcuts are loaded

### Reports not loading?
- Check your internet connection
- Refresh the page
- Check browser console for errors (F12)

### Activity feed empty?
- Activity logging starts from v0.8.0 forward
- Historical actions are not tracked
- New actions will appear automatically
